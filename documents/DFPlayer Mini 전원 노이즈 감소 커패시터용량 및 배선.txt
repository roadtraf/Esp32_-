🔨 조립 순서 (권장)
Step 1: 커패시터 확인
1. 100uF 커패시터 극성 확인
   - 긴 다리 = + (양극)
   - 짧은 다리 = - (음극)
   - 또는 몸체에 - 표시

2. 다리 구부리기 (선택)
   - 브레드보드 사용 시: 그대로
   - 납땜 시: 다리 짧게 자르기
Step 2: 전원 배선
1. ESP32 5V → DFPlayer VCC (빨간선)
2. ESP32 GND → DFPlayer GND (검은선)
3. 100uF 커패시터:
   - (+) → VCC 라인에 연결
   - (-) → GND 라인에 연결
   ⚠️ 극성 주의!
Step 3: 신호 배선
1. DFPlayer TX → ESP32 GPIO17 (RX2) 직접 연결
2. ESP32 GPIO18 (TX2) → 1kΩ 저항 → DFPlayer RX
3. 1kΩ 저항 → GND (전압 분배)
Step 4: 스피커 연결
1. DFPlayer SPK_1 → 스피커 + (빨간선)
2. DFPlayer SPK_2 → 스피커 - (검은선)
   (극성 상관없지만 통일 권장)
🧪 테스트 (단계별)
1단계: 전원만
1. DFPlayer VCC, GND만 연결 (신호선 제외)
2. 전원 ON
3. DFPlayer LED 확인:
   - 빨강 LED 켜짐 = 정상
   - LED 없음 = 배선 확인
   - 연기/냄새 = 즉시 전원 OFF! 극성 확인!
2단계: SD 카드
1. 전원 OFF
2. SD 카드 삽입 (딸깍 소리)
3. 전원 ON
4. LED 짧게 깜빡임 = SD 카드 인식
3단계: 신호 + 스피커
1. 전원 OFF
2. 모든 배선 완료
3. 전원 ON
4. 테스트 코드 업로드
5. 시리얼 모니터: "DFPlayer Mini online!"
6. 스피커에서 음성 재생
