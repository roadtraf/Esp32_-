## 📋 SD 카드 사용 현황

### 소프트웨어에서 이미 구현됨

#### 1. SD_Logger.cpp/h
```cpp
// 기능:
- 사이클 로그 (/logs/cycle_log.csv)
- 에러 로그 (/logs/error_log.csv)
- 일일 리포트 (/reports/YYYYMMDD.txt)
- 설정 백업

// 초기화:
void initSD() {
  if (!SD.begin()) {
    Serial.println("[경고] SD 카드 없음");
    return;
  }
  // 디렉토리 생성
  SD.mkdir("/logs");
  SD.mkdir("/reports");
}
```

- 부팅 시 `/logo.bmp` 또는 `/logo.jpg` 로딩

## 🔌 TFT LCD 연결 (SD 카드 포함)

### ILI9488 핀 배치

```
TFT 모듈 (전면)          ESP32-S3
──────────────────────────────────
[디스플레이 480×320]

TFT 모듈 (후면)
├─ SPI 핀
│  ├─ VCC ──────────── 3.3V
│  ├─ GND ──────────── GND
│  ├─ CS ───────────── GPIO 15
│  ├─ RESET ────────── GPIO 17
│  ├─ DC ───────────── GPIO 16
│  ├─ MOSI ─────────── GPIO 13
│  ├─ SCK ──────────── GPIO 14
│  ├─ LED ──────────── 3.3V
│  └─ MISO ─────────── GPIO 12
│
├─ 터치 (XPT2046)
│  ├─ T_CLK ───────── GPIO 14 (공유)
│  ├─ T_CS ────────── GPIO 10
│  ├─ T_DIN ───────── GPIO 13 (공유)
│  └─ T_DO ────────── GPIO 12 (공유)
│
└─ SD 카드 슬롯 (Micro SD) ✅
   ├─ SD_CS ───────── GPIO 5 (별도)
   ├─ SD_MOSI ─────── GPIO 13 (공유)
   ├─ SD_MISO ─────── GPIO 12 (공유)
   └─ SD_SCK ──────── GPIO 14 (공유)
```

## 💾 필요한 것: Micro SD 카드만!

### 권장 사양
```
용량: 4GB ~ 32GB (8GB 권장)
규격: Micro SD (SDHC)
포맷: FAT32
속도: Class 10 이상
가격: 약 5,000 ~ 10,000원
```

### 파일 구조 (미리 준비)
```
SD 카드 루트/
├─ logo.bmp (또는 logo.jpg)  ← 부팅 로고 (선택)
├─ /logs/                     ← 자동 생성
│  ├─ cycle_log.csv
│  └─ error_log.csv
├─ /reports/                  ← 자동 생성
│  ├─ 20260204.txt
│  └─ 20260205.txt
└─ /backups/                  ← 자동 생성
   └─ config_backup.json
```

## 📝 SD 카드 사용법

### 1. 포맷
```
Windows: 내 컴퓨터 → SD 카드 → 포맷 (FAT32)
Mac: 디스크 유틸리티 → 지우기 (MS-DOS FAT)
```

### 2. 로고 파일 넣기 (선택)
```
루트에 logo.bmp 또는 logo.jpg 복사
크기: 480×320 픽셀 권장
```

### 3. 카드 삽입
```
TFT 패널 후면 슬롯에 삽입
방향: 금속 접점이 위쪽
```

### 4. 부팅 확인
```

  타입: SDHC
  용량: 7580 MB
/logs 디렉토리 생성
```
