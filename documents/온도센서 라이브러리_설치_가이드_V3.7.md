## 🔍 코드에서 사용 예시

### Sensor.cpp에서 사용

```cpp
// ================================================================
// Sensor.cpp  —  센서 읽기 및 필터링 (v3.5 - 온도센서 추가)
// ================================================================
#include "Config.h"
#include "Sensor.h"
#include <OneWire.h>             // ← 여기 사용!
#include <DallasTemperature.h>   // ← 여기 사용!

// DS18B20 온도 센서 초기화
OneWire oneWire(PIN_TEMP_SENSOR);           // 1-Wire 통신
DallasTemperature tempSensor(&oneWire);     // 온도 센서 객체

void initSensors() {
  // ... 다른 센서 초기화 ...
  
  // 온도 센서 초기화
  tempSensor.begin();
  Serial.print("[센서] DS18B20 온도센서 초기화... ");
  
  if (tempSensor.getDeviceCount() > 0) {
    tempSensor.setResolution(12);  // 12비트 해상도
    Serial.println("성공");
  } else {
    Serial.println("실패");
  }
}

void readSensors() {
  // ... 다른 센서 읽기 ...
  
  // 온도 읽기 (1초마다)
  if (config.tempSensorEnabled) {
    static uint32_t lastTempRead = 0;
    
    if (millis() - lastTempRead >= 1000) {
      tempSensor.requestTemperatures();
      delay(100);
      
      float temp = tempSensor.getTempCByIndex(0);
      
      if (temp > -100 && temp < 150) {
        sensorData.temperature = temp;
      }
      
      lastTempRead = millis();
    }
  }
}
```

## 📖 라이브러리 상세 정보

### OneWire 라이브러리

#### 주요 기능
- 1-Wire 프로토콜 통신
- DS18B20, DS18S20 등 지원
- 여러 센서 동시 사용 가능

#### 주요 메서드
```cpp
OneWire oneWire(pin);        // 1-Wire 핀 설정
oneWire.reset();             // 버스 리셋
oneWire.search(addr);        // 디바이스 검색
oneWire.read();              // 데이터 읽기
oneWire.write(data);         // 데이터 쓰기
```

#### 공식 페이지
```
GitHub: https://github.com/PaulStoffregen/OneWire
문서: http://www.pjrc.com/teensy/td_libs_OneWire.html
```

### DallasTemperature 라이브러리

#### 주요 기능
- Dallas/Maxim 온도 센서 제어
- 섭씨/화씨 변환
- 해상도 설정 (9~12비트)
- 여러 센서 주소 관리

#### 주요 메서드
```cpp
DallasTemperature sensors(&oneWire);   // 객체 생성
sensors.begin();                        // 초기화
sensors.getDeviceCount();               // 센서 개수
sensors.setResolution(12);              // 해상도 설정
sensors.requestTemperatures();          // 온도 측정 요청
sensors.getTempCByIndex(0);             // 온도 읽기 (°C)
sensors.getTempFByIndex(0);             // 온도 읽기 (°F)
```

#### 공식 페이지
```
GitHub: https://github.com/milesburton/Arduino-Temperature-Control-Library
```

## 🧪 테스트 예제

### 간단한 온도 읽기 테스트

```cpp
#include <OneWire.h>
#include <DallasTemperature.h>

// GPIO 14에 DS18B20 연결
#define ONE_WIRE_BUS 14

OneWire oneWire(ONE_WIRE_BUS);
DallasTemperature sensors(&oneWire);

void setup() {
  Serial.begin(115200);
  Serial.println("DS18B20 테스트 시작");
  
  sensors.begin();
  
  int deviceCount = sensors.getDeviceCount();
  Serial.printf("센서 개수: %d\n", deviceCount);
  
  if (deviceCount > 0) {
    sensors.setResolution(12);
    Serial.println("초기화 완료!");
  } else {
    Serial.println("센서 없음!");
  }
}

void loop() {
  sensors.requestTemperatures();
  float tempC = sensors.getTempCByIndex(0);
  
  Serial.printf("온도: %.2f°C\n", tempC);
  
  delay(1000);
}
```

#### 예상 출력
```
DS18B20 테스트 시작
센서 개수: 1
초기화 완료!
온도: 25.31°C
온도: 25.37°C
온도: 25.43°C
...
```

## 💡 추가 정보

### DS18B20 여러 개 사용

```cpp
// 여러 센서 주소 저장
DeviceAddress sensor1, sensor2, sensor3;

void setup() {
  sensors.begin();
  
  // 각 센서 주소 가져오기
  sensors.getAddress(sensor1, 0);
  sensors.getAddress(sensor2, 1);
  sensors.getAddress(sensor3, 2);
}

void loop() {
  sensors.requestTemperatures();
  
  float temp1 = sensors.getTempC(sensor1);
  float temp2 = sensors.getTempC(sensor2);
  float temp3 = sensors.getTempC(sensor3);
  
  Serial.printf("센서1: %.2f°C\n", temp1);
  Serial.printf("센서2: %.2f°C\n", temp2);
  Serial.printf("센서3: %.2f°C\n", temp3);
}
```

### 해상도별 정확도

| 해상도 | 변환 시간 | 정확도 |
|--------|----------|--------|
| 9비트 | 93.75ms | 0.5°C |
| 10비트 | 187.5ms | 0.25°C |
| 11비트 | 375ms | 0.125°C |
| **12비트** | 750ms | **0.0625°C** ← 기본값 |
