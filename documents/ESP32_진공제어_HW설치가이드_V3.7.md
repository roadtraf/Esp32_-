## ✅ 부품 적정성 검토

### 🟢 적정 부품 (문제 없음)

#### 1. ESP32-S3 (N16R8)
✅ **적합**: 16MB Flash, 8MB PSRAM - 충분한 용량
- LovyanGFX 그래픽 처리 가능
- 다국어 폰트 로딩 가능
- 44핀 충분 (사용: 약 20핀)

#### 2. 압력 센서 (XGZP6847A)
✅ **적합**: -100~0 kPa, 0.5-4.5V 출력
- 목표 압력 -60 kPa 측정 가능
- ESP32 ADC 범위 내 (3.3V 기준)

#### 3. 전류 센서 (ACS712 20A)
✅ **적합**: 펌프 정격 42W ÷ 12V = 3.5A
- 20A 센서로 충분 (여유 있음)
- 돌입전류 대응 가능

#### 4. 진공 펌프 (VN-C4)
✅ **적합**: -85kPa, 40L/min
- 시스템 부피 60cm³ 기준 200ms 배기 가능
- 흡착컵 4개 구동 충분

#### 5. 솔레노이드 밸브 (VT307V-6g1-02)
✅ **적합**: 3방향 밸브
- 진공 유지 / 대기 개방 가능
- 12V 구동 적합

### 🟡 주의 필요 부품

#### 1. TFT LCD (ILI9488, SPI)
⚠️ **주의**: SPI 통신 속도
- **권장**: HSPI 핀 사용 (GPIO 12~15)
- **주의**: 긴 케이블 (50cm) 시 노이즈 가능
- **대책**: 쉴드 케이블 사용 ✅ (이미 준비됨)

#### 2. 호스 (ID 8mm + ID 6mm)
⚠️ **검토**: 코드에서 ID 4mm 기준 계산
- **실제**: ID 8mm 사용 시 배기 시간 더 짧아짐 (좋음)
- **영향**: 계산된 200ms보다 빠를 수 있음
- **대책**: 실측 후 `vacuumOnTime` 조정

#### 3. 흡착컵 (DP30 × 4개)
⚠️ **확인**: 스프링 홀더 조립 필요
- **구성**: 컵 + M12-L53 홀더 + 브라켓
- **주의**: 씰 상태 확인 (누설 방지)

### 🔴 추가 필요 부품

#### 1. DS18B20 풀업 저항 ⚠️ **필수**
❌ **누락**: 4.7kΩ 저항 (DS18B20용)
- **현재**: 10kΩ 저항 세트만 있음
- **필요**: 4.7kΩ 저항 1개 추가
- **가격**: 200원
- **용도**: DATA 핀 풀업 (센서 동작 필수)

#### 2. SD 카드 모듈 (선택)
❓ **선택 사항**: 로깅 기능 사용 시
- **필요 여부**: 코드에 SD_Logger.cpp 포함
- **추천**: Micro SD 모듈 + 8GB 카드
- **가격**: 5,000원
- **기능**: 사이클 로그, 에러 기록

✓ 밸브 - 이경 커플러
✓ 각 흡착컵 연결부 (4곳)
```

## 🔨 조립 순서

### Phase 1: 전원부 설치 (30분)

1. **ATX PSU 고정**
   - 엔클로저 바닥에 나사 고정
   - 12V 레일 확인 (노란선/검정선)
   
2. **DC 벅 컨버터 장착**
   - DIN 레일에 고정
   - 입력: 12V, 출력: 5V 3A로 조정
   
3. **전해 커패시터 연결**
   - 12V 라인 양단에 4700µF × 2
   - 극성 주의!
   
4. **EMI 필터 설치**
   - 12V 출력단에 직렬 연결

### Phase 2: 제어부 설치 (40분)

1. **베이스 플레이트 준비**
   - 20×30cm 플레이트에 스페이서 고정
   - ESP32, MOSFET, 센서 위치 마킹
   
2. **ESP32-S3 장착**
   - M3 스페이서로 고정
   - USB 포트 접근 가능하게 배치
   
3. **센서 모듈 배치**
   - 압력 센서: 호스 근처
   - 전류 센서: 펌프 라인에 직렬
   - DS18B20: 펌프 근처 (5cm 이내)
   
4. **광커플러 모듈 (PC817)**
   - 입력측: 12V 센서 신호
   - 출력측: ESP32 GPIO

### Phase 3: 구동부 설치 (1시간)

1. **진공 펌프 고정**
   - L형 브라켓 사용
   - 진동 패드 (6×10cm) 하부 부착
   - 프레임 상단에 고정
   
2. **솔레노이드 밸브 장착**
   - L형 브라켓으로 고정
   - 펌프 근처 배치
   
3. **배관 작업**
   ```
   ① 펌프 출구 → [호스 8mm] → T피팅
   ② T피팅 → [호스 8mm] → 압력센서 (분기)
   ③ T피팅 → [호스 8mm] → 밸브
   ④ 밸브 → [호스 8mm] → 이경 커플러
   ⑤ 이경 커플러 → [호스 6mm] → 흡착컵 유닛
   ```
   
4. **흡착컵 조립**
   - DP30 컵 + M12-L53 스프링 홀더
   - 브라켓 3개에 4개 컵 분산 배치
   - L형 피팅으로 호스 연결
   
5. **호스 클램프 체결**
   - 모든 연결부에 클램프 (10개)
   - 누설 방지 확인

### Phase 4: 배선 작업 (1시간 30분)

1. **전원 배선 (12V)**
   ```
   ATX 12V ─┬─ EMI Filter ─┬─ MOSFET Ch1 (펌프)
            │               ├─ MOSFET Ch2 (밸브)
            │               ├─ 광커플러 입력측
            │               └─ 부저
            │
            └─ Buck 5V ────┬─ ESP32
                            ├─ 센서들
                            └─ 광커플러 출력측
   ```
   
2. **신호 배선**
   - 듀퐁 케이블 사용
   - 터미널 블록으로 중계
   - 케이블 타이로 정리
   
3. **TFT 디스플레이**
   - 쉴드 케이블 (12C, 50cm) 사용
   - SPI 핀 연결 확인
   - 전면 패널 컷아웃 후 고정
   
4. **DS18B20 온도 센서**
   ```
   ⚠️ 중요!
   VCC ────────────── 5V
   DATA ──[4.7kΩ]── VCC (풀업 필수!)
     └────────────── GPIO 14
   GND ────────────── GND
   ```

5. **절연 입력 (광커플러)**
   ```
   리미트 SW ─┬─ 12V
              └─ GND ─[PC817]─ GPIO 40
   
   광센서 ────┬─ 12V
              └─ GND ─[PC817]─ GPIO 41
   
   비상정지 ──┬─ 12V
              └─ GND ─[PC817]─ GPIO 39
   ```

### Phase 5: 최종 조립 (30분)

1. **케이블 정리**
   - 케이블 타이로 묶기
   - 케이블 글랜드로 외부 배선 인입
   
2. **DIN 레일 부착**
   - 10cm 레일 × 6개
   - 모듈들 정렬 고정
   
3. **엔클로저 닫기**
   - 접지 확인
   - 통풍 확인

  - 다이오드 방향 확인

① USB 연결 (PC)

   [센서] 센서 초기화 완료
   [센서] DS18B20 온도센서 초기화...
```

### 3단계: 센서 테스트

```
✓ 온도 센서
  - Serial에 온도값 출력 확인
  - 손으로 감싸서 온도 상승 확인
  
✓ 압력 센서
  - 대기압에서 0 kPa 근처
  - 센서 흡입구 막으면 음압 표시
  
✓ 전류 센서
  - 무부하: 0A 근처
  
✓ 스위치 테스트
  - 리미트 SW: ON/OFF 확인
  - 광센서: 손 대면 감지
  - 비상정지: 동작 확인
```

### 4단계: 구동부 무부하 테스트

```
⚠️ 주의: 호스 미연결 상태

① 진공 펌프 단독 테스트
   - MOSFET 직접 12V 인가
   - 동작음 확인
   - 진동 확인 (패드 효과)
   
② 솔레노이드 밸브 테스트
   - Position A/B 전환 확인
   - 작동음 확인
```

### 5단계: 시스템 통합 테스트

```
① 호스 연결
② 흡착컵 미부착 상태
③ 프로그램 업로드
④ 메인 화면 확인:
   온도: 실온 표시
   압력: 대기압
   전류: 0A
   
⑤ START 버튼 터치
   - 펌프 동작 확인
   - 압력 하강 확인 (-60 kPa 목표)
   - 전류 3~4A 확인
   - 온도 상승 모니터링
   
⑥ 진공 해제 확인
   - STOP 버튼 또는 자동 완료
   - 밸브 작동 (쉬익 소리)
   - 압력 복귀 (0 kPa)
```

### 6단계: 실제 작업 테스트

```
① 흡착컵 조립 완료
② 테스트 박스 준비 (5kg 이하)
③ 리미트 스위치에 박스 접촉
   → STATE: IDLE → VACUUM_ON
④ 압력 -60 kPa 도달 확인
   → STATE: VACUUM_HOLD (5초)
⑤ 박스 들어올리기
⑥ 광센서로 박스 제거 감지
   → STATE: WAIT_REMOVAL → COMPLETE
⑦ 다음 사이클 대기
```
