# 진공 시스템 배기 시간 분석

## 📋 시스템 사양

### 펌프
- **모델**: 소형 DC 진공 펌프
- **최대 진공도**: -85 kPa (게이지압)
- **유량**: 40 L/min
- **전원**: DC 12V, 42W (3.5A)
- **절대압**: 16.3 kPa (대기압 대비 84% 진공)

### 배관 및 흡착
- **호스**: 내경 4mm, 길이 1.4m
- **흡착컵**: DP30 × 4개 (직경 30mm)
- **총 시스템 부피**: 약 60 cm³ (0.06 L)

| 85% | 16.3 | -85.0 | *도달 불가* |

### 실제 배기 시간 (누설 + 저항 고려)

실제 시스템에서는 다음 요인들로 인해 **이론값의 2~3배** 시간이 소요됩니다:

1. **누설 (Leakage)**
   - 호스 연결부 미세 누설
   - 흡착컵 씰 불완전
   - O-링 노화

2. **유동 저항 (Conductance)**
   - 호스 L/D 비율: 350:1 (매우 가늘고 긴 호스)
   - 4mm 내경 → 유동 저항 큼
   - 곡관부 압력 손실

3. **아웃개싱 (Outgassing)**
   - 호스/흡착컵 재질에서 가스 방출
   - 고진공일수록 영향 증가

**실전 배기 시간 (작업용 -60kPa 기준):**
```
이론: 81 ms
실제: 160~240 ms (2~3배)
권장: 200 ms (안전 여유)
```

## ⚙️ 실무 권장 설정값

### Config.h 권장값

```cpp
// 타이밍 설정 (단위: ms)
config.vacuumOnTime    = 200;   // 진공 형성: 200ms
config.vacuumHoldTime  = 3000;  // 유지: 3초 (작업 종류에 따라 조정)
config.vacuumBreakTime = 700;   // 해제: 700ms
config.waitRemovalTime = 5000;  // 제거 대기: 5초

// PID 설정
config.targetPressure = -60.0;  // 목표: -60kPa (실용적)
config.pressureHysteresis = 5.0; // 히스테리시스: ±5kPa
```

### 작업 종류별 권장값

#### 1. 가벼운 박스 (< 5kg)
```
targetPressure = -40 kPa
vacuumOnTime   = 150 ms
vacuumHoldTime = 2000 ms
```

#### 2. 일반 박스 (5~10kg)
```
targetPressure = -60 kPa  ← 권장
vacuumOnTime   = 200 ms
vacuumHoldTime = 3000 ms
```

#### 3. 무거운 박스 (> 10kg)
```
targetPressure = -75 kPa
vacuumOnTime   = 300 ms  (여유 있게)
vacuumHoldTime = 4000 ms
```

## 🔬 진공도별 흡착력

흡착컵 DP30 × 4개의 이론 흡착력:

| 진공도 | 게이지압 (kPa) | 흡착력 (N) | 흡착력 (kgf) |
|--------|---------------|-----------|-------------|
| -40 kPa | -40 | **113 N** | **11.5 kg** |
| -60 kPa | -60 | **170 N** | **17.3 kg** |
| -75 kPa | -75 | **212 N** | **21.6 kg** |
| -80 kPa | -80 | **226 N** | **23.0 kg** |

**계산식:**
```
흡착력 = 진공압력 × 유효 면적
유효 면적 = π × (30mm/2)² × 4개 = 2827 mm² = 28.27 cm²
```

**실제 흡착력은 이론값의 60~80% 수준**
- -60 kPa 기준: 약 **10~14 kgf** 실제 흡착 가능

## ⚠️ 주의사항 및 팁

### 1. 완전한 진공은 불가능
- **이론**: -85 kPa까지 가능
- **실제**: -70~-80 kPa 도달 (누설로 인해)
- **권장**: -60 kPa 목표 (안정적이고 빠름)

### 2. 호스 최적화
현재 호스는 **L/D 비율이 350:1로 매우 높음** (저항 큼)

**개선 방안:**
- 내경 6mm로 변경 → 배기 시간 약 40% 단축
- 길이 1.0m로 단축 → 배기 시간 약 30% 단축
- 단, 작업 공간 제약 고려 필요

### 3. 누설 최소화
- 호스 연결부: 호스 클램프 사용
- 흡착컵: 정기적으로 씰 상태 점검
- 피팅: 테플론 테이프로 기밀 강화

### 4. 센서 보정
압력 센서는 **대기압 기준으로 자주 영점 조정** 필요
```
현재 대기압: 101.3 kPa
센서 영점: 매일 또는 주간 단위 교정
```

## 📊 실험 권장 절차

### Step 1: 기본 동작 확인
```cpp
config.targetPressure = -40.0;
config.vacuumOnTime = 200;
```
→ 가벼운 샘플로 흡착/해제 반복 테스트

### Step 2: 진공도 상승
```cpp
config.targetPressure = -60.0;
config.vacuumOnTime = 250;
```
→ 실제 박스로 테스트

### Step 3: 최적화
- 압력 센서 로그 분석
- vacuumOnTime 최소값 찾기
- holdTime은 작업 특성에 맞게 조정

### Step 4: PID 튜닝 (선택)
```cpp
config.pidKp = 1.5;
config.pidKi = 0.3;
config.pidKd = 0.1;
```
→ 압력 오버슈트 최소화
