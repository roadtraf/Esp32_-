## 🎙️ 음성 녹음 가이드

### 발성 원칙
1. **명료성**: 모든 단어를 명확하게 발음
2. **속도**: 보통보다 약간 느리게 (1.2배 속도)
3. **톤**: 전문적이면서도 친근하게
4. **강세**: 중요 단어는 강조 (예: "긴급", "즉시")
5. **감정**: 메시지 종류에 따라 적절한 감정 표현
   - 시스템: 중립적
   - 상태: 안정적
   - 에러: 긴급하지만 당황하지 않음
   - 유지보수: 사전 예방 느낌
   - 안내: 친절하고 도움되는 톤

### 녹음 환경
- **조용한 공간** (배경 소음 최소화)
- **마이크 거리**: 입에서 15-20cm
- **포맷**: WAV 또는 MP3
- **비트레이트**: 128kbps 이상
- **샘플레이트**: 44.1kHz

### 후처리
1. 노이즈 제거
2. 볼륨 정규화 (-3dB 정도)
3. 앞뒤 무음 구간 제거 (0.2초 정도만 남김)
4. MP3 변환 (128kbps)

## 🎵 TTS 사용 가이드

### Google Cloud TTS (권장)
```python
from google.cloud import texttospeech

client = texttospeech.TextToSpeechClient()

# 한글
synthesis_input = texttospeech.SynthesisInput(
    text="진공 제어 시스템을 시작합니다."
)

voice = texttospeech.VoiceSelectionParams(
    language_code="ko-KR",
    name="ko-KR-Wavenet-A",  # 여성 (A, B)
    # name="ko-KR-Wavenet-C",  # 남성 (C, D)
)

audio_config = texttospeech.AudioConfig(
    audio_encoding=texttospeech.AudioEncoding.MP3,
    speaking_rate=0.9,  # 속도 (0.9 = 느리게)
    pitch=0.0,
)

response = client.synthesize_speech(
    input=synthesis_input,
    voice=voice,
    audio_config=audio_config
)

with open("001_welcome.mp3", "wb") as out:
    out.write(response.audio_content)
```

### Naver Clova (무료)
```python
import requests

client_id = "YOUR_CLIENT_ID"
client_secret = "YOUR_CLIENT_SECRET"

url = "https://naveropenapi.apigw.ntruss.com/tts-premium/v1/tts"

headers = {
    "X-NCP-APIGW-API-KEY-ID": client_id,
    "X-NCP-APIGW-API-KEY": client_secret,
}

data = {
    "speaker": "nara",  # 여성: nara, 남성: jinho
    "volume": "0",
    "speed": "-1",  # -5 ~ 5 (느리게)
    "pitch": "0",
    "format": "mp3",
    "text": "진공 제어 시스템을 시작합니다."
}

response = requests.post(url, headers=headers, data=data)

with open("001_welcome.mp3", "wb") as f:
    f.write(response.content)
```
