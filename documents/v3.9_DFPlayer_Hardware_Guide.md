# v3.9 DFPlayer Mini 음성 알림 시스템 - 하드웨어 가이드

## 📦 필요한 부품

### 1. DFPlayer Mini 모듈
**스펙:**
- 제품명: DFPlayer Mini MP3 모듈
- 지원 포맷: MP3, WAV, WMA
- 전압: DC 3.2~5.0V
- 출력: 3W (스피커 직접 구동 가능)
- 통신: UART (9600 baud 기본)
- 메모리: MicroSD 카드 (최대 32GB)

**구매처:**
- AliExpress: $1-2
- Amazon: $5-10
- 국내 온라인: 5,000-10,000원

**추천 모델:**
- DFPlayer Mini V3.0 (안정성 우수)
- GD3200B 칩 버전 (MH2024K-24SS보다 안정적)

### 2. MicroSD 카드
**스펙:**
- 용량: 4GB ~ 32GB
- 클래스: Class 4 이상
- 포맷: FAT32
- 속도: 평범한 SD 카드면 충분

**주의사항:**
- 64GB 이상은 지원 안 됨
- SDHC만 지원 (SDXC 불가)
- 반드시 FAT32 포맷

### 3. 스피커
**옵션 1: 소형 스피커 (권장)**
- 임피던스: 4Ω 또는 8Ω
- 출력: 2-3W
- 크기: 40mm ~ 57mm 직경
- 가격: 1,000-3,000원

**옵션 2: 헤드폰/이어폰**
- 3.5mm 잭
- 스피커보다 작은 소리

**옵션 3: 앰프 + 대형 스피커**
- PAM8403 앰프 모듈 ($1)
- 더 큰 볼륨 필요시

### 4. 전선 및 커넥터
- Dupont 점퍼 와이어 (Female-Female) 7개
- 또는 납땜용 전선
- 2.54mm 헤더 핀 (선택)

### 5. 기타 (선택)
- 저항: 1kΩ × 2개 (RX 라인 분배용, 권장)
- 커패시터: 100uF (전원 노이즈 제거, 선택)
- 3.5mm 오디오 잭 (스피커 교체 편의)

## 🔌 배선 가이드

### 기본 배선 (간단)

```
DFPlayer Mini          ESP32-S3
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
VCC      ───────────→  3.3V 또는 5V
GND      ───────────→  GND
TX       ───────────→  GPIO17 (RX2)
RX       ───────────→  GPIO18 (TX2)
SPK_1    ───────────→  스피커 +
SPK_2    ───────────→  스피커 -
```

**전원 선택:**
- 3.3V: 조용한 환경용 (소음 적음)
- 5V: 큰 볼륨 필요시 (권장)

### 권장 배선 (안정성 향상)

```
DFPlayer Mini          ESP32-S3
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
VCC      ───────────→  5V
GND      ───────────→  GND
         ──[100uF]──→  (전원 노이즈 제거)
TX       ───────────→  GPIO17 (RX2)
RX       ──[1kΩ]────→  GPIO18 (TX2)
         ──[1kΩ]────→  GND
                        (전압 분배)
SPK_1    ───────────→  스피커 +
SPK_2    ───────────→  스피커 -
```

**왜 저항?**
- ESP32는 3.3V 로직
- DFPlayer RX는 5V 허용
- 저항 분배로 전압 보호 (선택)

### 앰프 사용 시 배선

```
DFPlayer Mini          PAM8403          스피커
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
VCC      ───────────→  VCC (5V)
GND      ───────────→  GND
DAC_R    ───────────→  RIN
DAC_L    ───────────→  LIN
                       ROUT    ──────→  R+
                       GND     ──────→  R-
                       LOUT    ──────→  L+
                       GND     ──────→  L-
```

## 🛠️ 하드웨어 조립

### 1. DFPlayer Mini 모듈 확인
```
핀 배열 (위에서 볼 때):

 VCC  RX  TX  
 [  DFPlayer  ]
 GND SPK2 SPK1

또는:

1  VCC    16 BUSY
2  RX     15 USB-
3  TX     14 USB+
4  DAC_R  13 ADKEY_2
5  DAC_L  12 ADKEY_1
6  SPK_1  11 IO_2
7  GND    10 GND
8  SPK_2   9 IO_1
```

### 2. 배선 순서
1. **전원 끄기** ⚠️
2. DFPlayer VCC → ESP32 5V
3. DFPlayer GND → ESP32 GND
4. DFPlayer TX → ESP32 GPIO17
5. DFPlayer RX → ESP32 GPIO18
6. 스피커 연결 (극성 무관)
7. MicroSD 카드 삽입 (딸깍 소리)
8. **전원 켜기**

### 3. 테스트
```cpp
// 간단한 테스트 코드
#include <DFRobotDFPlayerMini.h>
#include <HardwareSerial.h>

HardwareSerial mySerial(2);  // UART2
DFRobotDFPlayerMini myDFPlayer;

void setup() {
  Serial.begin(115200);
  mySerial.begin(9600, SERIAL_8N1, 17, 18);  // RX, TX
  
  if (myDFPlayer.begin(mySerial)) {
    Serial.println("DFPlayer Mini online!");
    myDFPlayer.volume(20);  // 볼륨 20/30
    myDFPlayer.play(1);     // 첫 번째 파일 재생
  } else {
    Serial.println("Unable to begin DFPlayer");
  }
}

void loop() {}
```

**예상 결과:**
- "DFPlayer Mini online!" 출력
- 스피커에서 음성 재생

## 🔧 문제 해결

### 문제 1: "Unable to begin"
**원인:**
- 배선 오류
- MicroSD 카드 없음
- 잘못된 UART 핀

**해결:**
1. 배선 재확인 (TX ↔ RX 확인!)
2. SD 카드 삽입 확인
3. 전원 재시작

### 문제 2: 소리 안 남
**원인:**
- 볼륨 0
- 스피커 고장
- 파일 포맷 문제

**해결:**
1. `myDFPlayer.volume(25);` 확인
2. 스피커 테스트 (다른 장치)
3. MP3 파일 재변환

### 문제 3: 파일 재생 안 됨
**원인:**
- 파일명 규칙 위반
- SD 카드 포맷 문제

**해결:**
1. 파일명을 001, 002로 변경
2. SD 카드를 FAT32로 재포맷
3. 파일을 순서대로 복사

### 문제 4: 노이즈
**원인:**
- 전원 노이즈
- 그라운드 루프

**해결:**
1. 100uF 커패시터 추가 (VCC-GND)
2. 짧은 전선 사용
3. 3.3V 대신 5V 사용
